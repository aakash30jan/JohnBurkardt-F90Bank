FAIR_DICE_SIMULATION
Simulation of Two Fair Dice

FAIR_DICE_SIMULATION is a FORTRAN90 program which simulates N games in which two fair dice are thrown and summed, creating graphics files for processing by gnuplot.

Usage:

fair_dice_simulation n
where
  • n is the number of times the 2 dice are thrown.
