PGMA_IO
Portable Gray Map (ASCII) Files
Read and Write Utilities

PGMA_IO is a FORTRAN90 library which reads and writes files in the ASCII PGM (Portable Gray Map) format for 2D grayscale graphics information.

There are many graphics programs, such as XV and GIMP, that can display images in this format, or translate them to other formats such as BMP, GIF, JPEG, MacPaint, X Window, and so on.

Related Data and Programs:

PBMA_IO, a FORTRAN90 library which handles the ASCII Portable Bit Map format.

PGMA, a data directory which contains examples of PGMA files.

PGMA_TO_PGMB, a C++ program which converts an ASCII PGM file to a binary PGM file.

PGMB_TO_PGMA, a C++ program which converts a binary PGM file to an ASCII PGM file.

PPMA_IO, a FORTRAN90 library which handles the ASCII Portable Pixel Map format.
