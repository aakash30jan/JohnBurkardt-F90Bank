RUN_96
2 CVT Basis Vectors
for INOUT_FLOW2 Data

This run works with 800 flow solutions from the INOUT_FLOW2 dataset. We subtract 5/3 of steady solution from 1-400, and 1/3 from 401 through 800. We DON'T normalize the flow solutions. We use mass matrix preconditioning.

  • cvt_run_96.inp, an input file that computes a CVT basis of size 2.
  • cvt_run_96.out, the output of the run.
  • gen_001.txt, a CVT basis vector.
  • gen_002.txt, a CVT basis vector.
