RUN_41 - A run of CVT_BASIS

Run #41 works with 500 flow solutions in the CAVITY region. We subtract 5/3 of steady solution from 1-250, and 1/3 from 251 through 500. We NORMALIZE the PDE solutions.

  • cvt_run_41.inp, an input file that computes the energy of a set of preprocessed data for cluster size 2.
  • cvt_run_41.out, the output of the run.
  • gen_001.txt, centroid #1.
  • gen_002.txt, centroid #2.
