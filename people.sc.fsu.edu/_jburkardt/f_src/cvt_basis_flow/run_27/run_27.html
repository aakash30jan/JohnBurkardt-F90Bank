RUN_27 - A run of CVT_BASIS

Run #27 is like run #25, but for cluster size 8.

  • cvt_run_27.inp, an input file that computes the energy of a set of preprocessed data for cluster size 8.
  • cvt_run_27.out, the output of the run.
  • gen_001.txt, centroid #1.
  • gen_002.txt, centroid #2.
  • gen_003.txt, centroid #3.
  • gen_004.txt, centroid #4.
  • gen_005.txt, centroid #5.
  • gen_006.txt, centroid #6.
  • gen_007.txt, centroid #7.
  • gen_008.txt, centroid #8.

The program VECTOR_PLOT was used to make pictures of the first four CVT basis vectors.

  • xy.txt gives the locations of the nodes.
  • gen_001_dir.eps, direction field for basis vector 1.
  • gen_001_vec.eps, flow field for basis vector 1.
  • gen_002_dir.eps, direction field for basis vector 2.
  • gen_002_vec.eps, flow field for basis vector 2.
  • gen_003_dir.eps, direction field for basis vector 3.
  • gen_003_vec.eps, flow field for basis vector 3.
  • gen_004_dir.eps, direction field for basis vector 4.
  • gen_004_vec.eps, flow field for basis vector 4.
