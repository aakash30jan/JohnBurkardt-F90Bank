RUN_78 - A run of CVT_BASIS

Run #78 works with 500 flow solutions in the T-Cell region. We subtract 5/3 of steady solution from 1-250, and 1/3 from 251 through 500. We DO NOT normalize the results. We use MASS MATRIX preconditioning.

  • cvt_run_78.inp, an input file that computes the energy of a set of preprocessed data for cluster size 5.
  • cvt_run_78.out, the output of the run.
  • gen_001.txt, centroid #1.
  • gen_002.txt, centroid #2.
  • gen_003.txt, centroid #3.
  • gen_004.txt, centroid #4.
  • gen_005.txt, centroid #5.
