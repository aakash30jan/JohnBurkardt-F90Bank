ASA007
Inverse of a Symmetric Positive Definite Matrix

ASA007 is a FORTRAN90 library which computes the inverse of a symmetric positive definite matrix, by Michael Healy.

ASA007 is Applied Statistics Algorithm 7.

The algorithm implemented here uses a compressed storage for both the matrix A and the factor U. This saves some storage, but can make computations a little awkward.

Languages:

ASA007 is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version

Related Data and Programs:

ASA006, a FORTRAN90 library which computes the Cholesky factorization of a symmetric positive definite matrix. A version of this routine is used by ASA007.

ASA047, a FORTRAN90 library which implements the Nelder-Mead minimization algorithm, and uses a version of ASA007 for the computation.

LAPACK_EXAMPLES, a FORTRAN90 program which demonstrates the use of the LAPACK linear algebra library.

LINPACK, a FORTRAN90 library which includes routines to compute the inverse of symmetric positive matrix.
