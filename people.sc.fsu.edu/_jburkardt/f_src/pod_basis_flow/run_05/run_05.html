RUN_05
A run of POD_BASIS

This run of POD_BASIS takes the INOUT_FLOW data, applies mass matrix preconditioning, and seeks 16 POD vectors.

  • run_05.inp, the input file to POD_BASIS;
  • run_05.out, the output of the run.
