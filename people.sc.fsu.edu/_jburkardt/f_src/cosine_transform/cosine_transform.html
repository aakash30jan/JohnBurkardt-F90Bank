COSINE_TRANSFORM
A Discrete Cosine Transform for Real Data

COSINE_TRANSFORM is a FORTRAN90 library which demonstrates some simple properties of the discrete cosine transform (DCT) for real data.

The code is not optimized in any way, and is intended instead for investigation and education.

SINE_TRANSFORM, a FORTRAN90 library which implements the discrete sine transform.

WALSH, a FORTRAN90 library which implements versions of the Walsh and Haar transforms.
