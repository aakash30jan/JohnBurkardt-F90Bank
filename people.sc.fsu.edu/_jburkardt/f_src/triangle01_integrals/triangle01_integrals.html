TRIANGLE01_INTEGRALS
Integrals Inside the Unit Triangle in 2D

TRIANGLE01_INTEGRALS is a FORTRAN90 library which returns the exact value of the integral of any monomial over the interior of the unit triangle in 2D.

The interior of the unit triangle in 2D is defined by 

        0 <= x
        0 <= y
             x + y <= 1

The integrands are all of the form 

        f(x,y) = x^e1 * y^e2
where the exponents are nonnegative integers.
