LOCAL_MIN_RC
Reverse Communication Function for Local Minimum, by Richard Brent

LOCAL_MIN_RC is a FORTRAN90 library which seeks a local minimum of a scalar function of a scalar variable, without requiring derivatives, or assuming the function is differentiable, using reverse communication (RC), by Richard Brent.
