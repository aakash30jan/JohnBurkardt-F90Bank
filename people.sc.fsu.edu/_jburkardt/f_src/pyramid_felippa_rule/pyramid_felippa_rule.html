PYRAMID_FELIPPA_RULE
Quadrature Rules for the Pyramid

PYRAMID_FELIPPA_RULE is a FORTRAN90 library which returns Felippa quadrature rules over the interior of the unit pyramid in 3D.

Languages:

PYRAMID_FELIPPA_RULE is available in a C version and a C++ version and a FORTRAN90 version and a FORTRAN77 version and a MATLAB version.
