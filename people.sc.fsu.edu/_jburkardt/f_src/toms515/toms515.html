TOMS515
K subsets of an N set

TOMS515 is a FORTRAN90 library which lists the subsets of size K selected from a set of size N, by Bill Buckles, Matthew Lybanon.

Languages:

TOMS515 is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Data and Programs:

SUBSET, a FORTRAN90 library which enumerates combinations, partitions, subsets, index sets, and other combinatorial objects.

Author:

Original FORTRAN77 version by Bill Buckles, Matthew Lybanon. FORTRAN90 version by John Burkardt.

Reference:
