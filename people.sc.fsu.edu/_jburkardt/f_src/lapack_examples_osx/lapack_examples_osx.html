LAPACK_EXAMPLES_OSX
Examples Using the MAC OSX Precompiled LAPACK Library

LAPACK_EXAMPLES_OSX is a FORTRAN90 program which demonstrates how to use the precompiled LAPACK library on a Macintosh running the OSX operating system, by using the compiler option "-framework veclib", or, on the more recent "El Capitan" operating system, "-framework Accelerate".
