R8CC
Compressed Column Matrix Utilities

R8CC is a FORTRAN90 library which carries out some basic linear algebra operations on R8CC matrices (real 64 bit precision Compressed Column (Colptr, Row, Value) format).

Languages:

R8CC is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
